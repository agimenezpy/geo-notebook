## Requisitos

### Software
- Anaconda 3 2019.07
- Python 3.7

### Módulos
- GDAL
- GEOS
- PyProj
- Shapely
- Fiona
- Geopandas
- ArcGIS API for Python

### Comandos
```bash
$ conda install gdal geos pyproj shapely fiona geopandas
$ conda install -c esri arcgis

```
